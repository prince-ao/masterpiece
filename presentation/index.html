  • Masterpiece is a full-stack cross-platform mobile application.
  • It could be thought of as a hybrid between a social media app and an ecommerce app, but is much more.
  • It aims to bring together artists, galleries, investors, and art enthusiasts seamlessly into a single mobile app.
  • It creates an efficient art market buy introducing an online convolutional neural network model.

Features

Full Stack

  • Masterpiece is fully connected end-to-end with a developed frontend, backend, database, and machine learning model.

Monorepo

  • All the code for this entire application, including this presentation, is located in one repository.
  • This enables cross-functional programming as well as simpler building, testing, and deploying.

Microservice

  • We have two distinct HTTP servers running two different languages
  • We chose this architecture because we wanted to write our machine learning model in python and our actual backend in Typescript

Cloud Application

  • With the execution of the frontend, our entire application deployed in the cloud
  • This makes are app more scalable and easier to maintain.

Machine Learning

  • We created a machine learning model called Patina.
  • The goal of the model is to create an efficient art market, but predicting the value of a paining.
  • This model helps investors and artists make more rational decisions with regard to the value of art pieces.

Pages

Homepage

  • Displays the posts of the users that the user follows ordered by most recent to least recent
  • Gives you the ability to visit individual pages and like images.
  • Implements pagination

Search

  • User can search for any other user
  • User can select a user out of the list of outputs and it will navigate to their homepage

Profile

  • Displays a user's profile
  • Allows users to upload an image and follow other users
